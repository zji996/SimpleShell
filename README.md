# SimpleShell
## Commands
cd - Change directory  
pwd - Present Working directory  
mkdir - Make a directory (Alerts if already exists)  
rmdir- Remove the directory (Alerts if no such file or directory)  
ls - List contents of pwd  
cp - Copy contents from one file to another  
rf - readfile - read a file and print it  
wf - writefile - write to a file and print it  
exit  
general commands available in UNIX shell
